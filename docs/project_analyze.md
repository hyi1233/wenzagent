# WenzAgent 项目分析报告

## 📋 项目概览

**项目名称**：WenzAgent  
**描述**：纯Dart库，用于LAN通信、RPC和AI Agent管理  
**版本**：1.0.0  
**Dart SDK要求**：^3.11.0  
**仓库**：https://github.com/lyming99/wenzagent

## 🏗️ 核心架构

### 1. **Agent模块** (`lib/src/agent/`)
- **IAgent接口**：定义AI Agent的完整生命周期和操作
- **AgentImpl**：具体实现，包含：
  - 对话适配器（支持LangChain集成）
  - 消息处理器和队列管理
  - 工具注册和权限控制系统
  - 会话和上下文管理

### 2. **LAN模块** (`lib/src/lan/`)
- **LanHostService**：局域网服务端实现
- **LanClientService**：局域网客户端实现
- 支持文件传输、消息广播、设备发现

### 3. **RPC模块** (`lib/src/rpc/`)
- 远程过程调用管理器
- 支持流式调用和自动重连
- 协议定义和配置管理

### 4. **工具系统** (`lib/src/agent/tool/`)
- **内置工具**：文件操作、命令执行、搜索等
- **权限管理**：敏感操作需要用户授权
- **工具注册机制**：支持自定义工具扩展

## 🔧 技术特点

1. **纯Dart实现**：不依赖Flutter，跨平台兼容
2. **模块化设计**：各模块职责清晰，易于维护和扩展
3. **异步编程**：全面使用Dart的Future和Stream
4. **流式处理**：支持实时消息流传输
5. **权限控制**：完善的工具执行权限管理
6. **多客户端支持**：引用计数和锁机制保证一致性
